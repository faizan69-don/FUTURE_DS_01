# FUTURE_DS_01
Business Sales Performance Analytics Project using Excel Pivot Tables and Data Visualization.
Business Sales Performance Analytics

Project Overview
This project analyzes business sales data using Excel Pivot Tables and data visualization techniques.

Objectives
- Analyze sales by region
- Analyze sales by category
- Identify top-selling products
- Study revenue trends

Tools Used
- Microsoft Excel
- Pivot Tables
- Charts

Key Insights
- West region generated the highest revenue.
- Technology was the highest revenue-generating category.
- Canon imageCLASS 2200 Advanced Copier was the top-selling product.
